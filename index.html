Codigos para la consola

mkdir
Es una orden de los sistemas operativos usada para crear un nuevo sub directorio o carpeta del sistema de archivos.
cd (ruta a la que queremos movernos)
Para movernos entre carpetas. Para movernos a la carpeta superior pondremos "cd..".
ls
Nos muestra un listado de los archivos y carpetas que hay en el directorio en el que estamos.
rm
Para borrar archivos
mv
Para cambiar archivos de nombre, tambien sirve para mover archivos si especificamos la ruta
cat
cat de concatenar, nos permite visualizar el contenido de un archivo de texto sin la necesidad de un editor. Para utilizarlo solo debemos mencionarlo jnto al archivo que deseamos visualizar
